# Heart Disease Prediction Using Machine Learning

## Overview
This project predicts the likelihood of heart disease using machine learning algorithms.

## Problem Statement
Early detection of cardiovascular disease can reduce mortality and improve healthcare outcomes.

## Dataset
- Source: Heart Disease Dataset
- Features: Age, Cholesterol, Blood Pressure, Chest Pain Type, etc.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Machine Learning Models
- Logistic Regression
- Random Forest
- Support Vector Machine

## Results
Random Forest achieved the best performance.
